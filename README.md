# Magru Tools / WebScannner /

## Installation  
First put your openroute api key at api directory - 
```bash
git clone https://github.com/PhiVault/WebScannner.git
cd WebScannner
bash install.sh
```

## Usage  

Run the scanner on a website:  

```bash
bash webs
```
